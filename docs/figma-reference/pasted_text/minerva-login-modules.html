O bundle UMD expõe `window.lucide.icons` como objetos de tuplas `[["tag", {attrs}], …]`. Renderize via JS: converta o nome kebab-case para PascalCase, construa o SVG manualmente a partir das tuplas com atributos `width`, `height`, `viewBox`, `fill:none`, `stroke:currentColor`, `stroke-width:1.75`, `stroke-linecap/linejoin:round`. Use `data-icon` e `data-size` como attrs nos elementos, e um script inicializador com retry (setInterval de 50ms, até 40 tentativas) para garantir que o bundle carregou antes de renderizar. ## Restrições - Não invente cores fora dos tokens - Não use emoji - Não use Inter ou Roboto como destaque — Minerva usa Inter como base, pode usar fonte serif elegante (ex: Fraunces) só no logotipo da sidebar - Não adicione seções/dados não especificados acima - Preserve o `data-screen-label` em cada frame (`01 · Login`, `02 · Módulos`) - Carregue Inter via Google Fonts (weights 400–800)
